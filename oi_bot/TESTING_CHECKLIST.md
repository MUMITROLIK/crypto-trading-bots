## 🎯 Проверка скоринга

**SHORT сигнал должен набрать >= 10 баллов:**
- Памп + OI в боковике + L/S + Funding = ~12 баллов (базовые)
- Footprint продавцы = +1 балл
- Стенка продаж = +1 балл
- **Итого:** до 21 балла максимум

**LONG сигнал должен набрать >= 7 баллов:**
- OI рост + SHORT лики + CVD = ~8 баллов (базовые)
- Footprint покупатели = +1 балл
- Стенка покупок = +1 балл
- **Итого:** до 16 баллов максимум

## 🐛 Возможные проблемы

### 1. Order Book API лимиты
**Симптом:** Ошибки 429 Too Many Requests

**Решение:** Увеличить `_CACHE_TTL` в orderbook.py с 30 до 60 секунд

### 2. Footprint пустой
**Симптом:** `fp_bias = None` всегда

**Причина:** CVD tracker не передаёт цену

**Проверка:**
```python
# В cvd_tracker.py должно быть:
data_store.add_trade("binance", symbol, qty, is_buy, price)
# НЕ:
data_store.add_trade("binance", symbol, qty, is_buy)
```

### 3. Volume Profile пустой
**Симптом:** `vol_zones = []`

**Причина:** Недостаточно данных в price_history (< 10 точек)

**Решение:** Подождать 2-3 минуты после старта бота

### 4. Стенки не отображаются
**Симптом:** Нет строк "🧱 Стенка продаж" в сообщении

**Причина:** 
- Нет крупных стенок (< $8K или < 3x среднего)
- Стенки далеко от цены (> 4%)

**Это нормально** — не на каждой монете есть крупные стенки
